Social links

Share group

A share group contains icons which permit user to share a page via social media, email or print.

Interaction specs

On click/tap on:

  • Facebook/Twitter/Linkedin icon, user is taken to a new browser tab, where social media's share screen is shown. Based on the type of social media channel as well as business requirements, certain share fields are pre-populated.
  • Email icon - user's default mail app opens.
  • Print icon - a print version of the page is shown in the print dialog.

Rules

Accessibility

Provide ALT tags for all icons e.g. Print, Share on Twitter, etc.
